# H012 - Human L2 Review Attachment Index V1

## Purpose
Provide a single attachment-order reference for the human hosted `L2` handoff so the reviewer can open the packet in the right sequence once the Drive URLs are pasted in.

## Use
Place this index alongside `H009`, `H010`, and `H011` in the Drive handoff folder.

If the packet is delivered as `.docx`, preserve this same order in the Word bundle.

## Recommended attachment order
1. `H005-l2-reviewer-assignment-packet.md`
2. `H007-l2-review-kickoff-note.md`
3. `H008-l2-reviewer-checklist-v1.md`
4. `H009-drive-facing-l2-review-handoff-v1.md`
5. `H011-reviewer-readme-v1.md`
6. `l2-hosted-execution-bundle-v1.md`
7. `l2-review-handoff-protocol.md`
8. `scoring-rubric-v1.md`
9. `review-pass-worksheet-template.md`
10. `blinded-review-worksheet-template.md`
11. `l2-hosted-review-summary-template.md`

## What this index is for
- it gives the reviewer a stable reading order
- it reduces the chance of opening the packet out of sequence
- it keeps the handoff procedural while the shared URLs are still being finalized

## Output location
- Save this file in the same Drive folder as the rest of the human `L2` handoff packet.

## Completion rule
This index is ready once the Drive links are pasted into `H010` and `H009`.
