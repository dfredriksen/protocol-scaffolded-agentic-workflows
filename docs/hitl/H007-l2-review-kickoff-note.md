# H007 - L2 Review Kickoff Note

## Purpose
Provide a short handoff note that can be sent to a distinct evaluator to start the hosted `L2` review without additional repo interpretation.

## Reviewer-facing message

Please perform the hosted Teaching benchmark `L2` review as a distinct evaluator.

This is a bounded methods-and-scoring review of the hosted benchmark matrix only. It is not a full journal peer review, and it is not a line-editing request.

Plain-language rule: grade the frozen run outputs against the rubric. Do not grade the prompt text itself, and do not give credit or penalties just because an AI says positive things about its own work. If a document is self-referential, ignore the self-reference and score the actual evidence.

Review target:
- `BP001` hosted runs
- `BP002` hosted runs

Estimated effort:
- approximately `3-5` hours for a careful pass

Use these materials first:
- open the reviewer `.docx` packet folder first
- `H008-l2-reviewer-checklist-v1.docx`
- `l2-hosted-execution-bundle-v1.docx`
- `l2-review-handoff-protocol.docx`
- `scoring-rubric-v1.docx`

Please do not edit any frozen run artifacts. Score from retained artifacts only and record uncertainty explicitly.
Do not use any LLM, AI assistant, or automated judge during this review. This exercise must be human graded.

Required outputs:
- one consolidated hosted review summary
- per-run review memos or equivalent notes
- completed worksheets
- reviewer identity and independence level

Output location:
- `l2-hosted-matrix` results folder or equivalent designated handoff location

Frozen run artifacts:
- use the `review-artifacts` folder inside the parent human `L2` bundle
- score only the six hosted run folders under `review-artifacts/BP001/` and `review-artifacts/BP002/`
- use `review-artifacts/optional-l2a-comparison/` only after independent scoring is complete

A completed Anthropic `L2A` model-review pass exists for the same hosted BP001/BP002 slice, but use it only as secondary context after your own independent scoring is complete. Do not let it anchor your initial judgments.

## Access note

- The packet may be delivered by Drive, shared folder, zip archive, or `.docx` handoff bundle.
- The review requirements do not depend on Google Drive specifically.

## Completion condition

The hosted `L2` step is complete when:
- review outputs exist under `docs/benchmarks/reviews/l2-hosted-matrix/`
- score agreements or disagreements are written down explicitly
- the reviewer states the independence level used

## Notes

- This note is intentionally short.
- `H008` is the shortest reviewer instruction surface.
- `H011` and `H012` provide the simplest packet entry order.
- Hosted `L2` does not imply local independent validation.
- Anthropic `L2A` is supportive secondary review, not a substitute for human `L2`.
