Minimum reviewer workflow:

1. Open the reviewer `.docx` packet folder.
2. Read the eight files listed above in order.
3. Use the scoring rubric and worksheet templates from the same `.docx` packet folder.
4. Score only the hosted `BP001` and `BP002` runs from frozen artifacts.
5. Open the six hosted run folders inside `review-artifacts/BP001` and `review-artifacts/BP002`.
6. Save completed worksheets, memos, and final summary into `l2-hosted-matrix`.

## Current rule

Human `L2` remains the real independence gate.
Anthropic `L2A` is supportive secondary review only.
Do not use any LLM, AI assistant, or automated judge for the human `L2` scoring pass.
This review must be human graded.
Grade the frozen outputs against the rubric, not the prompt text or any self-review language inside the outputs.
If the packet is self-referential, treat that as packaging noise and score the actual deliverable.

## Setup note

- Google Drive is allowed but not required.
- This handoff is complete once the reviewer can open the packet materials, follow the reading order, and save outputs to `l2-hosted-matrix`.
- Prefer the reviewer `.docx` packet folder over markdown files or repo paths.
- Keep filenames unchanged where practical so the attachment index and manifest remain aligned.
- For Drive-based review, use `review-artifacts` as the source of the frozen run evidence.
