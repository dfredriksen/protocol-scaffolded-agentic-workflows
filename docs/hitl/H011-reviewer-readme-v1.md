# H011 - Reviewer README V1

## Purpose
Provide one simple entry point for the human hosted `L2` reviewer.

## Start here

1. Open `H009-drive-facing-l2-review-handoff-v1.md`.
2. Use the packet location listed there. This may be a Drive folder, shared folder, zip archive, or `.docx` packet bundle.
3. Open the following files in this order:
   - `H005-l2-reviewer-assignment-packet.docx`
   - `H007-l2-review-kickoff-note.docx`
   - `H008-l2-reviewer-checklist-v1.docx`
   - `H011-reviewer-readme-v1.docx`
   - `H012-human-l2-review-attachment-index-v1.docx`
4. Follow `H008` exactly for the review steps.
5. Open the `review-artifacts` folder in the same parent `L2` bundle.

## Review scope

Review only the hosted matrix:

- `BP001` hosted runs
- `BP002` hosted runs

Do not review local/open-weight runs in this pass.

## Output requirement

Save all completed human `L2` outputs into the `l2-hosted-matrix` results location referenced in `H009`.

Minimum outputs:

- one consolidated hosted review summary
- per-run review memos or equivalent notes
- completed worksheets
- reviewer identity
- independence level

## Important rule

Anthropic `L2A` review exists only as optional post-scoring comparison.

Do not use `L2A` to anchor your initial judgments.
Do not grade the AI's self-praise or prompt commentary; grade the frozen run outputs themselves against the rubric.
If a document refers to itself, ignore that self-reference and evaluate the underlying work product.

## Access rule

- The review packet does not require Google Drive specifically.
- If you received `.docx` files, use those reviewer-facing copies as the primary handoff surface.
- If any location or filename is unclear, record the uncertainty rather than guessing.
- For Drive-based review, use `review-artifacts` for the six frozen hosted runs and optional `L2A` comparison material.

## Completion

The review is complete when:

- the required outputs are in the `l2-hosted-matrix` folder
- agreements or disagreements are explicit
- the reviewer identity and independence level are stated
