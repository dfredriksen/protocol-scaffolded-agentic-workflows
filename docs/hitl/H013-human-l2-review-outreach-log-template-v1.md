# H013 - Human L2 Review Outreach Log Template V1

## Purpose
Provide one place to record who was contacted for the hosted human `L2` review, when the packet was sent, and what response state the outreach is in.

## Use
- Create one row per candidate reviewer.
- Update the row when the packet is sent, declined, accepted, or goes stale.
- Keep this log aligned with `H009`, `H010`, `H011`, and `H012`.

## Status vocabulary
- `Not contacted`
- `Contacted`
- `Interested`
- `Declined`
- `Accepted`
- `No response`

## Completion status vocabulary
- `Pending`
- `Packet sent`
- `Review in progress`
- `Review complete`
- `Closed without review`

## Completion rule
This log is active once the first reviewer candidate is named and should remain the repo-side reference for outreach state until a human `L2` review is complete or the solicitation round is closed.
