# BP001 Run Retention

Store one subfolder per run under this directory.

## Suggested run folder name
- `YYYY-MM-DD-run01`

## Expected contents
- `baseline-prompt.txt`
- `baseline-plan.md`
- `taught-prompt.txt`
- `taught-plan-draft.md`
- `taught-validator-feedback.md`
- `taught-plan-final.md`
- `score-summary.md`
- `evaluator-worksheet.md`

## Retention rule
Do not overwrite prior runs. Add a new run folder for each execution.
