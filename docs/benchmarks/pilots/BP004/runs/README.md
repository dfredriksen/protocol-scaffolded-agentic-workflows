# BP004 Run Retention

Store one subfolder per run under this directory.

## Suggested run folder name
- `YYYY-MM-DD-run01`

## Expected contents
- `baseline-prompt.txt`
- `baseline-workflow-trace.md`
- `baseline-validation-notes.md`
- `taught-prompt.txt`
- `taught-workflow-trace.md`
- `taught-validation-notes.md`
- `stop-condition-log.md`
- `handoff-packet-index.md`
- `artifact-retention-list.md`
- `created-files-manifest.md`
- `score-summary.md`
- `evaluator-worksheet.md`

## Optional created artifact folders
- `baseline-created/`
- `taught-created/`

## Retention rule
Do not overwrite prior runs. Add a new run folder for each execution.
