# Journal Target Decision v1

Date: 2026-03-15  
Scope: primary journal target selection for the current manuscript

## Decision

The current primary journal target is:

- `Empirical Software Engineering`

## Why This Is The Right Target Now

The paper’s strongest current properties are:

- empirical benchmark structure
- repeated within-model comparison across hosted providers
- workflow and evaluation discipline
- clear interest in engineering practice for agentic systems

Those properties align better with an empirical software-engineering journal than with HCI, multiagent, or broad AI theory venues.

## Why Not Conference Submission

Current instruction and program posture:

- no conference submission path is being pursued
- the manuscript should be hardened for a journal-first route

So all subsequent preparation should assume journal expectations:

- fuller methods transparency
- stronger appendices
- cleaner construct discipline
- more complete accounting than a conference-length paper would require

## Why Not Another Primary Target

### IEEE Software

Viable alternate, but not primary because the current package is still more empirical-study-driven than practitioner-article-driven.

### IEEE Intelligent Systems

Viable alternate, but not primary because the paper’s main strength is engineering methodology rather than broad AI-systems framing.

### TOSEM

Too ambitious as the immediate target.

### AI & Society

Poor fit for the present manuscript shape.

## Required Preparation For The Selected Target

Before venue-specific packet assembly, the manuscript should complete:

- run-accounting transparency work
- retained-selection reporting clarification
- construct-language tightening around `teaching`
- scope narrowing around the hosted empirical spine
- numeric transcription verification for the source human `L2` memo
- final editorial packaging for the submission bundle

## Secondary Fallback Order

If later revision reveals that the paper can be made substantially more practice-facing than methods-facing, the fallback order is:

1. `IEEE Software`
2. `IEEE Intelligent Systems`
